# Hail Data Download Web App

##  Description
Static web application for downloading radar hail data by station and date.

##  Features
- Browse 5 radar stations (Banphue, Omkoi, Phimai, Rongkwang, Takhli)
- Filter files by date range
- Download .npz hail data files
- Responsive web interface
- 24/7 online access via GitHub Pages

##  Live Demo
Visit: https://yourusername.github.io/hail-data-web/

##  How to Update
1. Add new .npz files to appropriate station directories
2. Update files.json with new file information
3. Commit and push changes to GitHub
4. GitHub Pages will auto-update

##  Technologies
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (ES6+)
- GitHub Pages
